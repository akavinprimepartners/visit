# AKAVIN OS - Master Blueprint

## Project Overview
AKAVIN OS is an AI-powered Business Operating System for Digital Commerce, CRM, Catalogs, Trade Shows, and Enterprise Automation.

## Quick Start
1. Run `./scripts/setup.sh`
2. Run `docker-compose up -d`
3. Access: http://localhost:3000 (Web) | http://localhost:8000 (API)

## Tech Stack
- **Frontend**: Next.js 14 + React + TypeScript
- **Backend**: FastAPI + Python 3.11
- **Database**: PostgreSQL + JSON Storage
- **AI**: Ollama + spaCy + Transformers
- **OCR**: PaddleOCR + Tesseract
- **Search**: Elasticsearch + PostgreSQL FTS
- **Deployment**: Docker + Docker Compose
